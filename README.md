# Pro Fleet Care Simcoe County Website

This is the official website for Pro Fleet Care Simcoe County, a mobile rust control service provider serving the Simcoe County area.

## Features

- Responsive design for all devices
- Interactive chatbot assistant ("Rusty")
- Contact form for quote requests
- Detailed service information
- Industry-specific solutions
- Mobile-first approach

## Local Development

1. Clone the repository:
```bash
git clone https://github.com/LearnHD/profleetcare-simcoe.git
cd profleetcare-simcoe
```

2. Open `index.html` in your browser to view the site locally.

## Structure

- `index.html` - Main homepage
- `pages/` - Additional website pages
- `css/` - Stylesheet files
- `js/` - JavaScript functionality
- `images/` - Website images and assets

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts
